# Business Requirements Document — Gunung Andong Landing Page

## 1. Problem Statement
Calon pendaki pemula kesulitan membayangkan jalur pendakian Gunung Andong secara visual dan naratif. Informasi tersebar di berbagai forum dan blog dengan format tidak konsisten.

## 2. Solution
Landing page satu halaman yang menyajikan alur pendakian secara visual (timeline) dan naratif (cerita tiap basecamp), dengan data praktis (ketinggian, durasi, fasilitas).

## 3. Target Metrics
| Metrik | Target |
|--------|--------|
| Page load time | < 2 detik |
| Mobile usability | 100% |
| Bounce rate | < 40% |
| Scroll depth | > 80% |

## 4. Value Proposition
- **Untuk pendaki:** Panduan lengkap dalam satu halaman
- **Untuk admin:** Mudah diupdate (static site)
- **Biaya:** Nol (Cloudflare free tier + static assets)

## 5. Success Criteria
- User dapat memahami jalur Andong hanya dengan scrolling
- Informasi basecamp mudah ditemukan
- Mobile experience setara desktop
