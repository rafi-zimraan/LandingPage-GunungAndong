# Product Requirements Document — Gunung Andong Landing Page

## 1. Tujuan
Landing page untuk mendokumentasikan perjalanan pendakian Gunung Andong dari BC1 (Desa Sawit) hingga BC5 (Puncak 1.726 mdpl), menargetkan pendaki pemula.

## 2. Target Audiens
- Pendaki pemula (18-35 tahun)
- Pencinta alam yang belum pernah ke Andong
- Pembaca mobile-first

## 3. Fitur (MVP)

### Wajib (P0)
- [x] Hero Section — tagline + CTA scroll
- [x] Stats Bar — ringkasan numerik
- [x] Timeline 5 Basecamp — narasi + info praktis
- [x] Tips Pendakian — 6 kartu tips
- [x] Tabel Ringkasan — data jalur
- [x] CTA Akhir — tombol navigasi
- [x] Mobile responsive
- [x] Scroll animation fade-in

### Sebaiknya (P1)
- [ ] Google Maps integration (API key)
- [ ] Galeri foto interaktif
- [ ] Dark mode toggle

### Nanti (P2)
- [ ] Multi-language (EN)
- [ ] Blog/news section
- [ ] Weather widget Andong

## 4. Non-Functional Requirements
- Lighthouse Performance ≥ 90
- Mobile-first responsive design
- Zero JavaScript errors
- SEO meta tags
- Build size minimal
